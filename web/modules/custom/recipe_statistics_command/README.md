## INTRODUCTION

The Recipe Statistics Command module provides a custom Drush command to generate and display statistical insights related to recipe content within the Drupal site.

The primary use case for this module is:

Generating recipe-related statistics via the command line

Assisting administrators and developers with reporting and data analysis

Enabling automation or scheduled execution of recipe metrics

## REQUIREMENTS

This module requires the following Drupal core and contributed modules:

Drupal 10.x

## INSTALLATION

Install as you would normally install a contributed Drupal module.
See: https://www.drupal.org/node/895232 for further information.

## CONFIGURATION
Enable the Recipe Statistics Command module

Run the provided Drush command to generate recipe statistics

(Optional) Integrate the command into cron jobs or CI/CD pipelines
