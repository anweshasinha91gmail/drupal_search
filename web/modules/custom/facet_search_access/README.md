## INTRODUCTION

The Facet Search Permission module provides fine-grained access control over Search API facets based on user permissions and roles.

The primary use case for this module is:

Restricting visibility of specific facets for certain user roles

Allowing only authorized users to interact with sensitive or internal facets

Managing facet access without duplicating views or search pages

## REQUIREMENTS

This module requires the following Drupal core and contributed modules:

Drupal 10.x

Search API

Facets

## INSTALLATION

Install as you would normally install a contributed Drupal module.
See: https://www.drupal.org/node/895232 for further information.

## CONFIGURATION
Enable the Facet Search Permission module

Assign the provided facet access permissions to user roles

Configure facet visibility based on permissions in the facet settings
